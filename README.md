# Online Test Portal

A web-based Online Test Portal developed using HTML, CSS, and JavaScript. The application allows users to register, log in, take tests, and view their results through a simple and user-friendly interface.

## Features

### User Authentication
- User Registration
- User Login
- Password Validation
- Session Management

### Online Examination
- Multiple Choice Questions (MCQs)
- Randomized Questions
- Timer-Based Test
- Automatic Score Calculation

### Result Management
- Instant Result Display
- Correct Answer Evaluation
- Score Tracking

### User-Friendly Interface
- Responsive Design
- Easy Navigation
- Clean UI
